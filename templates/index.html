Wireguard KeyGen Tool

Thank you for using the wireguard key generator tool.

Instructions

JSON Output

In the url, please enter the number of keys that you would like.

For Example: If you would like to generate three keys, then place in the url: /3

Data will come out as a JSON format.

CSV Output

If you desire a csv output then use /csv/<Number of keys>

For Example: If you would like to generate three keys, then place in the url: /csv/3
